# Bagula.ai Website - Quick Start Guide

## Installation

```bash
cd /Users/anupsingh/projects/bagula-workspace/bagula.ai/packages/website
npm install
```

## Development

```bash
npm run dev
```

Then open http://localhost:3000 in your browser.

## Production Build

```bash
npm run build
npm start
```

## What You Get

A complete premium landing page with:

- Hero section with navigation
- Problem statement (black box challenges)
- 4 key features (Cost, Performance, Quality, Regression)
- How It Works (3-step process)
- Social proof section
- Pricing (Self-hosted & Cloud)
- Professional footer

## Design Features

- Clean neutral color palette (stone/slate grays + blue accents)
- Inter font family
- Fully responsive (mobile, tablet, desktop)
- Tailwind CSS for styling
- TypeScript for type safety
- Production-ready code

## File Organization

- `/app` - Next.js pages and layouts
- `/components` - Reusable UI components
- `/public` - Static assets (add your illustrations here)

## Customization

1. **Content**: Edit component files in `/components`
2. **Colors**: Modify `tailwind.config.ts`
3. **Fonts**: Update in `app/layout.tsx`
4. **Images**: Add to `/public` directory

## Tech Stack

- Next.js 14 (React 18)
- TypeScript 5
- Tailwind CSS 3
- Inter Font
