## Layout Patterns

### Two-Column Pattern (Used in Features)
```
Left: Content (tag, title, description, list)
Right: Illustration or visualization
Alternates: Every other feature reverses order
```

### Card Grid Pattern (Used in Problems, Steps)
```
2-column grid on desktop
1-column on mobile
Equal height cards
Consistent padding
```

### Center-Aligned Pattern (Used in Headers)
```
Centered text
Max-width constraint
```
