# Bagula Enterprise Strategy

## Open Source + Cloud Business Model

Bagula follows a successful open-source business model similar to platforms like Conductor OSS, Supabase, and GitLab. The core testing framework is 100% open-source (Apache 2.0), while we offer a managed cloud platform for enterprises.

---

## Open Source (Free Forever)

### What's Included

- **Full Testing Framework** (`@bagula/core`)
  - Test runner with baseline comparison
  - Cost and latency monitoring
  - Confidence calibration
  - Multi-provider support (OpenAI, Anthropic, custom)

- **CLI Tool** (`@bagula/cli`)
  - Run tests locally
  - Generate reports (JSON, HTML, Markdown)
  - Baseline management
  - CI/CD integration

- **Integrations** (`@bagula/integrations`)
  - LangChain
  - CrewAI
  - AutoGPT
  - Microsoft Semantic Kernel

- **CI/CD Plugins** (`@bagula/plugins`)
  - GitHub Actions
  - GitLab CI
  - Jenkins
  - CircleCI

### Philosophy

- **No feature gatekeeping** - Core functionality is fully open
- **No artificial limits** - Run unlimited tests locally
- **No phone-home tracking** - Your data stays with you
- **Community-driven** - Accept contributions, transparent roadmap

## Competitive Advantages

### vs. In-House Solutions
- **Time to value**: Days vs months
- **Maintenance**: Zero vs ongoing
- **Expertise**: Built-in vs learning curve
- **Features**: Complete vs basic

### vs. Generic Testing Tools
- **AI-specific**: Built for agents, not web apps
- **Probabilistic**: Handles non-determinism
- **Cost-aware**: Tracks token usage
- **Baseline tracking**: Detects subtle regressions

### vs. LLM Observability Tools
- **Testing-first**: Proactive, not just reactive
- **CI/CD native**: Block bad deploys
- **Budget gates**: Prevent cost explosions
- **Open-source**: No vendor lock-in
